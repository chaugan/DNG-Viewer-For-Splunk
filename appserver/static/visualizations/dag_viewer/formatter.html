The DAG Viewer visualization renders Directed Acyclic Graphs from GraphViz DOT format.

Data Format:

  • Raw DOT mode: Return a field named "dot" containing the complete DOT graph string
  • Structured mode: Return fields "source", "target", and optionally "label" to build edges automatically

Note: Values in your DOT string take precedence over these options. These options are only applied if not specified in your DOT.
